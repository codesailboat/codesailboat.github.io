14_最长公共前缀

解题思路①

  • 按列进行比较，以第一行为基准，当第一行某列和其他行该列字符不相同时，前面几列即为最长公共前缀。将第一行该列字符置为'\0'(字符串结束标志)，而后return

  • 程序开始应该判断字符串数组是否为空，可以判断长度是否为零，为零return空字符串

  • 程序结束应该写return，因为当第一行被其他行包含时，循环中的return并不会执行

代码

char * longestCommonPrefix(char ** strs, int strsSize){
    int i;
    int j;
    if( strsSize == 0)
    {
        return "";
    }
    for(j=0; *(*strs+j) != '\0'; j++ )  //列
    {
        for( i=1; i<strsSize; i++)      //行
        {
            if( *(*strs+j) != *(*(strs+i)+j) )
            {
                *(*strs+j) = '\0';
                return *strs;
            }
        }
    }
    return *strs;
}
