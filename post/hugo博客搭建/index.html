Hugo博客搭建

一、软件安装

1.安装hugo

  • GitHub官网的Hugo Releases下载对应版本
    Hugo Releases： https://github.com/gohugoio/hugo/releases
  • windows版本已经是成品，直接下载解压，在环境变量path里添加文件夹路径即可。注意不要直接新建，正确流程如下:
    变量path -> 编辑 -> 新建 -> 选择文件夹路径

2.安装git

  • 官网下载较慢，可自己百度下载

二、本地创建博客

1.创建站点

  • 打开命令行（win+R，输入cmd),移动目录到希望保存博客网页的文件夹
  • 命令行中输入 hugo new site myblog
    其中 myblog 为文件夹名，myblog文件夹 即为博客根目录

2.创建博客

  • 移动到myblog目录下，命令行中输入: hugo new first.md
    first为文件名
  • 用markdown语言进行博客编写

3.下载并设置博客主题

  • 主题库地址： https://themes.gohugo.io
  • 命令行目录移至博客根目录，命令行中输入对应路径进行克隆，具体信息可在主题下方直接复制即可。
    以m10c主题为例，：
    复制输入： git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c
    注: 执行此操作之前确保git成功安装并配置成功，在命令行初始目录下输入 git 有输出即配置成功。

4.本地启动博客

  • 博客根目录下，在命令行中输入：
    hugo server -t m10c --buildDrafts
Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
Press Ctrl+C to stop
  • 浏览器中输入：http://localhost:1313/ 可以看到博客界面

三、本地博客部署到远端服务器

1.在GitHub上注册账号，新建仓库

  • 仓库名称为用户名.github.io，且全部小写。

2.部署仓库

  • 博客根目录下输入：hugo --theme=m10c --baseUrl="https://用户名.github.io" --buildDrafts
    该步骤完成后博客目录下生成了public文件夹
  • 进入public目录: cd public
  • 输入git init
  • 输入git add *
  • 输入git commit -m "我的hugo博客第一次提交"
  • 输入git remote add origin https://github.com/用户名/用户名.github.io.git
  • 该步骤若提示 fatal: remote origin already exists. ,输入移除命令git remote rm origin再重新上传。

3.上传文件

  • 输入git push -u origin master
  • 输入用户名和密码，此时GitHub仓库里可以看到文件。
  • 打开网页输入网址：用户名.github.io ，可以看到博客网页。

hugo博客的其他事项

  • 主题修改：在根目录下的config.toml文件中修改，第一次需要添加，例如：theme = "mainroad"

  • 上传内容：最终上传到GitHub的只是public目录下的内容。如果使用GitHubDesktop不可以把整个文件拷贝到本地库中，只是文件中的内容

  • 博客部署到GitHub时，仓库最好是空的，否则可能不允许合并到主干，需要另外新建分支，再进行上传
