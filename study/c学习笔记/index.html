C 学习

1. extern “C”

  • ‘extern “C” ‘和’__cplusplus’的主要作用就是都是为了实现C++与C兼容,使C++代码能够正确调用C语言代码。加上extern “C"后，会指示编译器这部分代码按C语言的进行编译。c++中定义了__cplusplus，C语言中没有该定义。
#ifdef __cplusplus
extern "C" {
#endif

void *memset(void* ,int , size_t);

#ifdef __cplusplus
}
#endif
